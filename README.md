# Automated Report Generation

## Project Overview
This project reads data from a CSV file, performs basic data analysis, and automatically generates a formatted PDF report with visualizations.

## Objective
Develop a Python script that:
- Reads data from a file
- Analyzes the data
- Generates a formatted PDF report
- Includes charts and summaries

## Technologies Used
- Python
- Pandas
- Matplotlib
- ReportLab

## Features
- Reads sales data from CSV
- Calculates total sales
- Calculates total profit
- Identifies best-selling product
- Creates graphical visualization
- Generates PDF report automatically

## Files Included
- `report_generator.py` → Main script
- `sales_data.csv` → Input dataset
- `sales_chart.png` → Generated chart
- `sales_report.pdf` → Sample generated report

## Output
The script generates:
- Sales analysis summary
- Visualization chart
- Automated PDF report
